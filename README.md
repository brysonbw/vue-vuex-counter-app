# Vue Counter App

Simple [vue](https://v2.vuejs.org/) [(vue cli v2)](https://cli.vuejs.org/guide/installation.html) + [vuex](https://vuex.vuejs.org/) counter app (example)

### Usage
```
npm install
```

### Developing
```
npm run dev

to view in browser open http://localhost:3000/ 
```

### Build
```
npm run build
```

### Lint
```
npm run lint
```
